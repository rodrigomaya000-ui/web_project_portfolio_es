# 🌐 Portafolio Web – RM

## 📌 Descripción

Portafolio web personal donde presento los proyectos desarrollados durante mi formación en desarrollo web. Este sitio refleja mis habilidades en HTML y CSS, con enfoque en diseño responsivo, uso de Flexbox y buenas prácticas de maquetación.

---

## 🚀 Demo

👉 [Ver portafolio en línea](https://rodrigomaya000-ui.github.io/web_project_portfolio_es/)

📌 Sitio desplegado con GitHub Pages
📱 Diseño responsivo optimizado para pantallas desde 320px hasta 4K

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Diseño responsivo (Responsive Design)

---

## 🎯 Objetivo del proyecto

El objetivo de este proyecto es:

- Practicar la estructura semántica en HTML
- Aplicar estilos con CSS de forma organizada
- Implementar layouts usando Flexbox
- Adaptar el diseño a múltiples resoluciones (desktop-first)
- Construir una base sólida para futuros proyectos con JavaScript

---

## 📱 Características

- Diseño completamente responsivo (320px → 4K)
- Layout flexible usando Flexbox
- Estructura semántica en HTML
- Adaptación a diferentes dispositivos (desktop-first)
- Código organizado y escalable
